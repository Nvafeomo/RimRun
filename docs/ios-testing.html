Test RimRun on iPhone / iPad

RimRun beta builds are distributed through TestFlight. Install Apple’s TestFlight app first, then accept our invite.

TestFlight invite — We’ll email you an invite or share a public link when the build is on TestFlight.

IOS_TEST_LINK_PLACEHOLDER

Before you start

  • iPhone or iPad on a recent iOS version.
  • Apple ID signed into the App Store.
  • TestFlight installed from the App Store.
  • Email address added as a tester in App Store Connect (internal) or invited via TestFlight link.

Install steps

  1. Install TestFlight if you don’t have it.
  2. Open the invite link we sent (email or the button above once set) on your iPhone.
  3. TestFlight opens → tap Accept, then Install for RimRun.
  4. Launch RimRun from TestFlight or your home screen.
  5. Allow location for the court map; allow photos only if you add a profile picture.

What to try

  • Sign up or sign in (email, Apple, or Google).
  • Browse courts on the map and open a court detail.
  • Subscribe to a court and post in court chat.
  • Send feedback to rimrun.support@gmail.com (include iOS version and what you were doing).

Troubleshooting

  • No invite showing — Check the Apple ID on your device matches the email we invited.
  • Apple / Google sign-in errors — Try again on Wi‑Fi; report the exact message if it keeps failing.
  • Blank map — Settings → RimRun → Location → While Using the App.
